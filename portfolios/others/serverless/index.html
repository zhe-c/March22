This Azure Function project is base on NSCC Course Map

  • Get All Academic Years: https://nscccoursemapapi.azurewebsites.net/api/academicyears

  • Get One Academic Year: https://nscccoursemapapi.azurewebsites.net/api/academicyears/{id}

  • Get All Courses:https://nscccoursemapapi.azurewebsites.net/api/courses

  • Get One Course: https://nscccoursemapapi.azurewebsites.net/api/courses/{id}

  • Get All Diploma Programs:https://nscccoursemapapi.azurewebsites.net/api/diplomaprograms

  • Get One Diploma Program: https://nscccoursemapapi.azurewebsites.net/api/diplomaprograms/{id}

  • Get All Instructors:https://nscccoursemapapi.azurewebsites.net/api/instructors

  • Get One Instructor: https://nscccoursemapapi.azurewebsites.net/api/instructors/{id}

  • Get All Semesters:https://nscccoursemapapi.azurewebsites.net/api/semesters

  • Get One Semester: https://nscccoursemapapi.azurewebsites.net/api/semesters/{id}

  • Get One Semester2: https://nscccoursemapapi.azurewebsites.net/api/semesters2/{id}

Please replace {id} with a number. If the number is out of range, it will return a 404.
